§ 04 · CORE · WHAT MAKES THIS PROJECT 04 / 08

The core scoring formula picks one path per query.

The retrieval core has one binary decision: free path (BM25 + RRF) or paid path (vector + rerank). The user opts into the paid path with a single env flag. Source: src/state/hybrid-search.ts.

BM25 · per-term score
score(D, t) = IDF(t) · tf(t, D) · (k₁ + 1) / (tf(t, D) + k₁ · (1 - b + b · |D| / avgdl))
k₁ = 1.2, b = 0.75. Source: src/state/hybrid-search.ts.
Reciprocal Rank Fusion · merge two rankings
RRF(d) = Σᵢ 1 / (k + rankᵢ(d))
k = 60. Source: src/state/search-index.ts.
